# SQL Helpdesk Reporting Portfolio

## Overview
This repository contains a sample SQL support database and reporting queries designed to demonstrate practical SQL skills for IT support and systems environments.

## Business Scenario
The database simulates a helpdesk ticketing environment where users log technical issues and technicians are assigned to resolve them.

## Database Objects
- Departments
- Users
- Technicians
- Tickets

## Included Files
- `schema/create_tables.sql` - table creation script
- `data/insert_data.sql` - sample data inserts
- `queries/open_tickets.sql` - query for open tickets
- `queries/overdue_tickets.sql` - query for unresolved older tickets
- `queries/tickets_by_department.sql` - ticket count by department
- `queries/technician_workload.sql` - technician workload report

## Skills Demonstrated
- Database design
- Table creation
- Data insertion
- Joins
- Aggregation
- Filtering
- Reporting queries

## Why This Project Matters
SQL is widely used in systems support, application support, ERP support, and reporting roles. This project demonstrates the ability to structure data and produce useful reports from operational systems.
