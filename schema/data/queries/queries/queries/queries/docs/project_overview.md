# Project Overview

This project simulates a simple IT helpdesk database for support and reporting purposes.

## Business Scenario
An organization needs to track support tickets raised by users across different departments. Technicians are assigned to resolve issues, and management needs reports on ticket volume, technician workload, and unresolved issues.

## Tables
- Departments
- Users
- Technicians
- Tickets

## Reporting Goals
- View open tickets
- Identify overdue unresolved tickets
- Count tickets by department
- Measure technician workload
